Pre-Fetching and De-Duplication Implementation for Coda File System

C, C++ • February 2014 - March 2014

Implemented dynamic pre-fetching for files based on analysis and intelligent prediction techniques in Coda which is a widely used distributed file system. More information about Coda

This greatly improved performance of Coda especially for environments featuring high latency and low bandwidth. I integrated pre-fetching in Coda client and de-duplication feature into Coda server.
